# Assignment 1 — Introduction to Data Mining & EDA
**Course: Data Mining & Discovery | University of Arizona**

## 📌 Objective
Introduction to Python for data mining — installing libraries, creating 
variables, calculating statistical metrics, and performing basic Pandas 
operations on the Fatal Police Shootings dataset.

## 📊 Dataset
`fatal-police-shootings-data.csv` — included in this folder.

Key features: `id`, `name`, `date`, `manner_of_death`, `armed`, `age`, 
`gender`, `race`, `city`, `state`, `signs_of_mental_illness`, 
`threat_level`, `flee`, `body_camera`

## 🔑 Key Tasks
- Installed and imported core data science libraries (Pandas, NumPy, 
  Matplotlib)
- Created DataFrames and performed basic Pandas operations
- Calculated statistical metrics: mean, median, mode, standard deviation
- Explored dataset structure — shape, dtypes, value counts per column
- Identified unique values and distributions across key features

## 🛠️ Tools & Technologies
`Python` `Pandas` `NumPy` `Matplotlib` `Jupyter Notebook`
