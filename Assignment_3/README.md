# Assignment 3 — Data Preprocessing & Feature Engineering
**Course: Data Mining & Discovery | University of Arizona**

## 📌 Objective
Perform comprehensive data preprocessing including missing value 
handling, feature scaling, data transformation, correlation analysis, 
and sampling on the Fatal Police Shootings dataset.

## 📊 Dataset
Fatal Police Shootings dataset — see Assignment 1 for source file.

## 🔑 Key Tasks
- Handled missing values (NaN) using appropriate imputation strategies 
  per column
- Applied feature scaling techniques (StandardScaler, MinMaxScaler)
- Performed data transformations for normalization
- Computed and visualized correlation matrix to identify feature 
  relationships
- Applied sampling techniques for data reduction
